# SmartDailyApps Website

Marketing website for SmartDailyApps.

Live site: https://smartdailyapps.com

## Repo Purpose

This repository contains the public studio website for SmartDailyApps,
including the home page, contact page, and legal pages.

## Stack

- Static HTML
- Tailwind via CDN
- GitHub Pages / GitHub Actions deployment

## Notes

- The public site domain is configured via `CNAME`.
- This repo is for website content only and does not host the MyGoldFolio app code.
- Product-specific changelog and version tracking live in the `mygoldfolio-website` repo.
